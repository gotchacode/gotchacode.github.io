VMware Player,Best in Class

Nov 27th, 2011

If you are Advanced Linux/Unix User and testing latest OS is what you enjoy, then you should definitely switch to VMware Player. Here is the link where you can download it. Once downloaded you can run the downloaded .bundle file using this command in your terminal ($sh ./filename.bundle ) . Make sure your PWD is same as that of the downloaded file. Once installed you are good to go. It is a hassle free software and in many case is better than all its counterparts.
