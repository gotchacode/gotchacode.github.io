Install Linux a Easy Way…without Getting Into Trouble :)

Aug 17th, 2010 | Comments

hey guys…many of u must have heard of linux and even got urself one of the free cd but got scared away because of troubles in partitioning and many of you must have ended up screwing your windows installation.But there is a very easy solution to this problem.

well here i assume the reader to have a zero knowledge of linux.

well the first thing u need to do is to install DAEMON TOOLS….HERE IS THE LINK  #mce_temp_url

DOWNLOAD AND INSTALL DAEMON TOOL.

the next thing u need to do is to download cd image of UBUNTU OR LINUX MINT.you can get these cd images free and easily from dere sites…guess u can easily google out for them.

now i will give u screenshots of how u can install linux on your own without getting into any trouble

[caption id=”attachment_107” align=”alignleft” width=”950” caption=”mount the cd image of the OS.”][/caption]

here we mount the cd image of the OS that is ubuntu or linux mint. how do we do that, we will mount the cd image from the location where you have downloaded the image.

[caption id=”attachment_108” align=”alignright” width=”950” caption=”Mount the OS to be installed in our case UBUNTU”][/caption]

next the autoplay starts …its will prompt you to click on WUBI.

[caption id=”attachment_109” align=”alignright” width=”950” caption=”Autoplay window of the UBUNTU.click on WUBI.exe”][/caption]

Now as you click on it,a new windows opens…dere you need to specify the destination drive. the size of harddisk  as you can see in the  screenshot below. you can also specify your password for security and privacy concerns.

[caption id=”attachment_110” align=”alignright” width=”950” caption=”insert the password.HDD size in this window and click on install”][/caption]

SO guys you must be thinking it very easy to install and you know what THIS LINUX INSTALL AS IF ITS LIKE ANY OTHER APPLICATION LIKE PICASA….you can remove this anytime by going into controlpanel>add/remove programs…THIS LINUX INSTALLS INSIDE WINDOWS SO THERE IS NO NEED OF ANY PARTITION.
